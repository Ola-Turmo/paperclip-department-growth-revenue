# Growth Revenue Department — Research Directory

This directory contains research briefs, benchmark notes, and upstream findings for the Growth Revenue Department.

## Purpose

Per the PRD's continuous learning requirements, this directory captures:
- Research briefs on funnel stages, attribution issues, and segment behavior
- Benchmark notes from experiments and historical performance
- Upstream findings from cross-department analysis

## Structure

```
docs/research/
  ├── attribution/
  │   └── [attribution analysis briefs]
  ├── funnel/
  │   └── [funnel diagnostic briefs]
  ├── segments/
  │   └── [segment intelligence briefs]
  └── experiments/
      └── [experiment learning briefs]
```

## Guidelines

- Each research brief should include: date, topic, findings, confidence level, and follow-up questions
- Update briefs when new data reveals changes to assumptions
- Cross-reference with /docs/adrs/ when decisions are made based on research
- Reference /evals/ for validation of research findings
